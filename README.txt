Project ini berisi script otomatisasi menggunakan Puppeteer yang disimpan di folder ui-test dengan nama file test.js. Script ini dibuat untuk melakukan Quality Assurance (QA) sederhana pada website Saucedemo dengan alur login, navigasi produk, menambahkan item ke cart, dan membuka keranjang belanja. Setiap langkah diberi jeda agar prosesnya terlihat jelas dan cocok untuk direkam sebagai video demo. Untuk menjalankan project ini, pastikan Node.js sudah terpasang, lalu lakukan instalasi dependency dengan npm install. Setelah itu, script dapat dijalankan menggunakan perintah node ui-test/test.js dan browser akan terbuka secara otomatis untuk memperlihatkan alur QA. Project ini ditujukan sebagai contoh penggunaan Puppeteer untuk testing otomatis, dokumentasi visual, maupun pembelajaran dasar tentang automasi browser.
